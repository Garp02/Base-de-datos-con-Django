## Objetivo
Creación de una base de datos con acceso Web usando Django en el lenguaje de programación Python. 
---

### Instrucciones 
Con ayuda del framework Django crear y gestionar una base de datos SQLite con el 
lenguaje de programación Python.

1. La base de datos tendrá información relevante de las 4 computadoras más
rápidas del mundo:

| Campo | Descripción |
|-------|-------------|
| **Nombre** | Nombre de la supercomputadora |
| **País de origen** | País donde se encuentra |
| **Número de cores** | Número total de núcleos |
| **Cantidad de RAM (TB)** | RAM disponible en terabytes |
| **Almacenamiento (TB)** | Capacidad de almacenamiento |
| **TeraFLOPS** | Rendimiento en TFLOPS |
| **Sistema operativo** | Sistema operativo utilizado |

2. Funciones:
  a. Agregar
  b. Editar
  c. Borrar
  d. Visualizar

3. Verificar el buen funcionamiento de lo que se pide en el proyecto.
